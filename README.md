☁️ Atmosfera: Next-Gen SwiftUI Weather Experience
Atmosfera is more than just a weather app; it’s a dynamic atmospheric simulation that brings the sky to your fingertips. By leveraging the full power of SwiftUI, we’ve transformed weather data into a living, breathing digital experience.

✨ Key Features
🎭 Immersive UI/UX

Dynamic Aura Engine: A shifting "Glow" background that adapts its color palette based on current weather conditions.

Live Weather Overlays: High-performance real-time rain and snow particle systems that make you feel the elements.

Glassmorphism Aesthetic: Sleek, modern card layouts utilizing Apple's ultraThinMaterial for a premium frosted-glass look.

🕹️ Interactive Depth

Spatial Parallax Physics: Integrated with CoreMotion, the interface responds to your device's physical tilt, creating a stunning 3D depth effect.

Animated Smart Modules:

Wind Dynamics: An interactive wind card with an icon that spins relative to real-time wind speeds.

Solar Journey: A visual sun-path tracker showing exactly where the sun is in the sky.

Contextual Progress Bars: Visualizing temperature ranges and "feels-like" deltas at a glance.

🧠 Intelligent Logic

Expandable Forecasts: Smoothly animated sections for hourly and daily projections.

AI-Powered Insights: Context-aware style and activity recommendations based on the current forecast.

System-Wide Harmony: Fully adaptive to iOS Light/Dark modes with elegant, automated transition states.

🛠️ Technical Stack
Language: Swift 5.10+

Framework: SwiftUI

Architecture: MVVM (Model-View-ViewModel)

Data Source: OpenWeatherMap API

Hardware Sensors: CoreMotion (Gyroscope & Accelerometer)

Networking: URLSession & Async/Await

🚀 Getting Started
Clone the repo: git clone https://github.com/YOUR_USERNAME/Atmosfera-Weather.git

Open the project in Xcode 15+.

Navigate to WeatherViewModel.swift.

Insert your OpenWeatherMap API key into the apiKey variable.

Hit Run and experience the atmosphere!

Design Philosophy: "Form follows function, but emotion leads the way." — This project was built to prove that utility can be beautiful.
